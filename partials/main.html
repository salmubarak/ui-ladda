UI.Ladda

AngularJS directive for Bootstrap 3 Ladda UI.

Overview

Bootstrap 3 Laddui was made by @msurguy. The original UI concept is by @hakimel which merges loading indicators into the action that invoked them. It is primarily intended for use with forms where it gives users immediate feedback upon submit rather than leaving them wondering while the browser does its thing.

See the original website for more information: http://msurguy.github.io/ladda-bootstrap/

Usage

Load the ui-ladda.js script in your application:

<script src="ui-ladda.js"></script>

Add the ui-ladda module as a dependency to your application module:

angular.module('myApp', ['ui.ladda']);

Apply the directive to your buttons:

<button class="btn btn-primary ladda-button" data-ui-ladda="progress" data-style="expand-right"> <span class="ladda-label">expand-right</span> </button>
